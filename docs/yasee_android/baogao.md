# SDK 性能报告
--- 


以下的测试均以:
- 环境区分 ( 恶劣、正常)
- 测试设备数为 4**台设备**
- 测试使用手机 **3台 (华为、OPPO、荣耀)**
- 每台设备 测试次数 **20** 次
- 每个环境 重连次数 **10** 次
- 平均测算 全部去掉 最高 最低秒数
