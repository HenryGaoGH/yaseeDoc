## 脉诊

### 指令信息

:::info
具体如何获取支持的交互指令; 可查阅 [👉 获取指令列表](comm_third_api)
:::

脉诊检测项支持如下交互指令:
1. 开始测量(左)
2. 开始测量(右)
3. 终止测量
4. 获取结果
5. 复位
6. 波形开关

| 指令名称 | 作用 |
|:----:|:----:|
| 波形开关 | 控制波形数据发送与否;建议根据自由业务控制; !长时间的开启会增加手机耗电 |
| 开始测量(左右手) | 控制测量左手还是右手   |
| 终止测量 | 测量期间中止、测量完成中止等行为   |
| 获取结果 | 测量完毕之后获取结果使用   |
| 复位 | 测量之前需要将设备统一复位操作   |

### API 字段解释

| 字段 | 含义 |
|:----:|:----:|
| codeIndex | 第几段加压; 左右手分别有 0~5   |
| cunG | 寸压力值   |
| guanG | 关压力值   |
| chiG | 尺压力值 |
| cun | 寸波点数据   |
| guan | 关波点数据   |
| chi | 尺波点数据 |
| code | 错误代码 |
| desc | 错误详情 |
| niu | 检测结果(已清洗) |
