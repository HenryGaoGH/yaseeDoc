## 概要说明
线上常驻分支:
- main (Prd作用)
- dev  (开发使用)
- test (测试使用)

非 常驻分支:
- 修复线上问题的分支
- 每个人的 迭代分支 

每次的线上问题修复, 都会自动同步到dev
