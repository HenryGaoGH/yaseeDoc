# Yasee SDK 支持的功能清单
--- 

## **⬇️目前的检测目标⬇️**
### 蓝牙设备的搜索 & 筛选 & 绑定 & 连接 & 交互,包含:
- [x] TMD设备 (Y906、911、916、917)
- [x] 沃莱设备 (MY_SCALE、iconmon、Y900) **转发数据**
- [ ] 玉成设备 (T9、M5、T20)

### TMD 设备支持的检测项
- [x] 血压
- [x] 尿酸
- [x] 血糖
- [x] 血酮
- [x] 血氧
- [x] 尿液
- [x] 肺功能
- [x] 血脂
- [x] 体温
- [x] 荧光
    - [x] C 反应蛋白
- [ ] 糖化血红蛋白


### 沃莱 设备支持的检测项
- [x] 体脂 (8电极 & 4电极)


### 玉成 设备支持的检测项
- [ ] 心电检测
- [ ] 动态体温检测


### 脉诊 设备支持的检测项
- [ ] 脉诊检测
